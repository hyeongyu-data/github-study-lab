# GitHub Study Lab

Git과 GitHub를 직접 손으로 익히기 위한 실습 저장소입니다.

## Study Roadmap

1. Git 기본기: 저장소, 변경사항, 커밋
2. 브랜치와 병합: 실험 공간 만들기
3. GitHub 원격 저장소: push, pull, clone
4. Pull Request: 리뷰와 협업 흐름
5. Issues와 Projects: 작업 관리
6. Actions 기초: 자동화와 CI

## Today's Goal

- 로컬 Git 저장소를 만든다.
- 파일 변경사항을 확인한다.
- 첫 커밋을 만든다.
- GitHub에 올릴 준비 흐름을 이해한다.
